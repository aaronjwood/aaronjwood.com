Period Killer

Period Killer is a free and open source directory and filename cleaning tool written in C#. It is built with simplicity, speed, and efficiency in mind.

How It Works

After launching Period Killer, simply select a folder that contains the folders to be processed and click "Clean Folder Names". This will replace all periods found in directory names with a space. Additionally, if you want files to be processed and have their periods replaced check the box under File -> Options that says "Process Filenames". If you want a portion of the directory and/or file name's text to be removed, check the box under File -> Options that says "Remove Folder Text" and/or "Remove Filename Text" and enter in the text. This will make the program remove the text you entered (and everything after it) from each directory and/or filename it processes.
