Text Analyzer

Text Analyzer is a free tool for analyzing text files. It can show you the words contained in the text file, find a word, or show the frequency of each word in the text file.

How It Works

Once the program is running, choose the "Analyze Text" option to select a text file that you want analyzed. The "Map Type" options are completely optional and are only used to show performance differences between the two data structures. If you wish to see the performance differences, simply select the map type that you want to use before selecting the text file you want to analyze.
